# strans
  CJK input method

## Switch Key
| Key | Action |
|-----|--------|
| Ctrl+N | Hiragana |
| Ctrl+K | Katakana |
| Ctrl+S | Hangul |
| Ctrl+T | English |
| Ctrl+E | Emoji(☺,λ,ε) |

## Dependencies
- plan9port
- libxcb
